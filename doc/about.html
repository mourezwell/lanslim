Historical
  • V0.1
    • (2008/02/25)
    Initial Version
     
  • V0.2
    • (2008/10/03)
    New features :
    - Double-click on contact to start a simple talk (head to head)
    - Contact drag and drop to invite them in current talk
    - Hyperlink management
    - Import/Export contacts
    - System tray icon management
     
  • V0.3
    • (2008/03/19)
    New features :
    - Auto line break in message area
    - Talk export
    - Talk local rename
    - Contact sorting in table
    - Talk members Tooltip on talk tab header
    - Group contact export
    - Refresh button to force availability update
    - Contact filters
     
  • V0.4
    • (2008/03/25)
    Fixes various issues
     
  • V0.5
    • (2008/04/02)
    Initial Source Forge Version
    New Features :
    - About dialog
    - No more notification of system message
     
  • V0.6
    • (2008/06/15)
    New features :
    - Contacts may be organized into categories thanks to the new Tree view
    - Position and Size are automatically saved and restored on application start
    - Minimize button has been added in main toolbar (it acts as tray if enabled)
    - Simplification of closing action settings
    - Improvement of the contact drag'n'drop
    - Improvement of contact list (or tree) udpates in case of remote contact update
    - Control added in order to avoid settings modification during talks
    - Control added in order to avoid name conflict between settings and online contacts
    Fixes some small display issues.
     
  • V0.61

    - Fix People In Update when closing talk
    - Sort Members by Name in Group Frame
