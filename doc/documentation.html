Answers
  1. How to install Lanslim ?

    2. Once you have downloaded the archive file of the windows executable, you just have to unzip the file and launch the excutable. To enable icon tray minimization you have to keep executable and dynamic library (dll) in the same directory.
     
  2. How to configure Lanslim ?

    3. Once you have launched the application, you have to click on the Settings Button and fill the two mandatory fields Name and Host.
    On this dialog you also can set your default style, configure shortcuts, import/export contacts, define the contact view display, customize the contact drag and drop, choose your language  
  3. How to configure Contacts ?

    4. There are two ways to configure contact. Either you can import contact file through the settings dailog as descrided in     How to import/export contacts ?. Or you can create contact via button , then you have to fill the two mandatory fields Name and Host, and eventually choose a category.
    Note : you can also create groups as described in How to create a Group ? or categories as described in How to create a Category ?
     
  4. How to start a new talk ?

    5. Heare again there are many ways to start a new talk : Either you use the "New Talk" button and choose a title and the members of the talk among your online contacts and groups. Or you double-click on a online Group or User contact.
     
  5. How to use message history ?

    6. Once you have started a talk you can callback the last ten messages you have sent by using the combinaison of keys "Shift + Up" and "Shift + Down" to browse your history
     
  6. How to use shortcuts ?

    7. Via the settongs dialog you can define shortcuts, that enable to add comonly used sequence of text by pressing Function keys F1 to F12
     
  7. How to send multiline message ?

    8. To send a message you have to click on send button or just press the Enter key, so if you want to write multi-line message you have to use the following combination of keys "Alt + Enters"
     
  8. How to import/export contacts ?

    9. Contacts may be exported/imported via csv files. These fucntions are available form Settings dialog via buttons and
     
  9. How to create a Group ?

    10. You can create a group via the new group button
     
  10. How to create a Category ?

    11. You can create a group via the new category button
     
  11. What is the difference between categories and groups ?

    12. Groups are a subset of your contacts with who you can start a talk quickly via double click
    Whereas categories are just a way to organize your contact in the contact tree
     
  12. How to restore a profile ?

    13. If you open the application and no contacts appears in your contact list, it may be because the last saving of your profile failed, try to use your bakup profile in your personnal directory (Commonly c:/doc and Setting/UserName under windows). To use it copy it then rename it.
     
  13. How to invite or exclude people from talk ?

    14. To invite people you can drag contact from "Contact view" to "Who's In? view" or to the talk pane. But you also can click on invite button and select in the dialog the contacts you want to invite.
    To exclude people you have to select them in the "Who's In? view" and click on the exclude button
     
  14. How to export a talk ?

    15. If you want to keep a trace of your talk, you can export the text via right click on the tab of the talk tab pane and select the output file on dialog.
     
  15. What is the use of different buttons ?

    16. Icon Use
    Display the About dialog with contact and version.
    Minimize the main frame (if tray is enable, hide the application even from taskbar)
    Exit the application
    Send the current message
    Start a New Talk
    Configure application settings
    Configure font Color
    Configure font style (Italic or not) (Ctrl + i)
    Configure font style (Bold or not) (Ctrl + b)
    Configure font style (Underline or not) (Ctrl + u)
    Create user
    Create group
    Delete user
    Edit user/group/category
    Refresh Availability
    Create category

     
  16. What is the display of different availabilities ?

    17. There are three different availabilities : ONLINE , OFFLINE and UNKNOWN used for groups
     
  17. How to add a new language ?

    18. You can open the excutable with winzip and find the message files (one in french, one in english) You have to translate the messages present in this file and send me the translation. I will create a new version taking this new language into account
